# Profile: Local Development (MacBook)

This profile is for local development machines with GUI and full tooling.

## Environment Context

- Interactive terminal with GUI access
- Browser available for testing
- Git push allowed after confirmation
- Docker Desktop available

## Relaxed Rules

- Can run `npm run dev` / `bun dev` without confirmation
- Can open browser for testing
- Can use IDE integrations freely

## Still Requires Approval

- Deployment to any environment
- Modifying production configs
- Running scripts that cost money (API calls, cloud resources)
